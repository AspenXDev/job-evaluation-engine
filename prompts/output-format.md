Output MUST be valid JSON.

Structure:

{
"job_title": "",
"company": "",
"scores": {
"scope_clarity": 0,
"psychological_load_risk": 0,
"technical_match": 0,
"operational_structure": 0
},
"confidence": "",
"evidence": {
"scope_clarity": [],
"psychological_load_risk": [],
"technical_match": [],
"operational_structure": []
},
"summary": ""
}

No additional commentary outside JSON.
