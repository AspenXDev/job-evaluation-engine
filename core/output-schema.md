# Output Schema

The engine produces structured output in the following format:

{
"role_summary": "",
"candidate_fit_score": 0,
"dimension_scores": {
"technical": 0,
"experience": 0,
"domain": 0,
"culture": 0,
"communication": 0,
"risk": 0
},
"strengths": [],
"gaps": [],
"risk_flags": [],
"recommendation": ""
}

All outputs must strictly adhere to this schema.
